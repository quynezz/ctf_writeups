**Description:** Using netcat (nc) is going to be pretty important. Can you connect to fickle-tempest.picoctf.net at port 54458 to get the flag?

**Topic:** General Skills 

Run the `nc` command to establish TCP connection in the required port and domain

```bash 
nc fickle-tempest.picoctf.net 54458 
```
And we have the flag: 

**Flag:** `picoCTF{nEtCat_Mast3ry_0d33dA2C}`
